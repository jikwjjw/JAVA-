+ 一维数组转二维数组
```
       int[][] ans = new int[list.size() / 2][2];
        for (int i = 0; i < list.size(); i++) {
            int num = list.get(i);
            //一维数组nums中的第index个除二取余数为二维数组的行
            int a = i / 2;
            //一维数组nums中的第index个除二取模为二维数组的列
            int b = i % 2;
            ans[a][b] = num;
        }
```
